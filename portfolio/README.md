# Portfolio Website

This portfolio website is built as a modern dark-themed landing page for a developer profile.

## What’s included

- `index.html` — homepage structure and sections
- `styles.css` — dark luxury theme, responsive layout, animated cards
- `script.js` — mobile navigation and section reveal logic

## Adding Screenshots

To add screenshots for your projects:

1. Take screenshots of your project demos or interfaces.
2. Save them in the `screenshots/` folder with the following names:
   - `motorbike-portal.png` - Screenshot of Luxury Motorbike Portal
   - `ai-bot.png` - Screenshot of Smart AI Information Bot
   - `virtual-eye.png` - Screenshot of Virtual Eye Obstacle System
   - `finance-system.png` - Screenshot of Zorvyn Finance System
   - `image-gallery.png` - Screenshot of Interactive Image Gallery
   - `angular-admin.png` - Screenshot of Role-Based Admin Portal
   - `codsoft-data.png` - Screenshot of Codsoft Data Science Work

3. Ensure images are high-quality and representative of your projects.
4. Recommended size: 800x600 pixels or similar aspect ratio.

The CSS is already configured to display these images as backgrounds with gradient overlays for better text readability.

## Deploy

Recommended platforms:
- GitHub Pages
- Netlify
- Vercel

Just push this folder to a repository and connect the platform to deploy a live portfolio site.
